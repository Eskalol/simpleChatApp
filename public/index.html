    About this project

    The purpose of this project is to learn react, foundation and socket.io. The site is built with gulp, and served as an express application. I thought jade might would be usefull, untill I understood that I was supposed to write html inside the react render function. The jade template became pretty small because of that, and the only thing the body contains is a #chat-app tag, hmm... not quite as planned. The site is still under development. I'm also going to test out the cool new library motion ui in this project.
